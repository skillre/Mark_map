Markdown思维导图API客户端

API使用说明

本页面演示如何使用API将Markdown转换为思维导图。

API端点

/api/convert

请求方法

POST

请求体格式

{
  "markdown": "# 你的Markdown内容"
}

响应格式

{
  "root": { /* 思维导图数据结构 */ },
  "features": { /* 特性标记 */ }
}

提示： API支持所有Markdown格式，包括标题(#)、列表(- +)、加粗、斜体等。
